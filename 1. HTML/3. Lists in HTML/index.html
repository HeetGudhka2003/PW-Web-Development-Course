Lists in HTML

There are three types of lists: Ordered List and Unordered List and Nested Lists

Ordered list: Numbers, Letters, Roman

  1. Item 1
  2. Item 2
  3. Item 3
  1. Item A
  2. Item B
  3. Item C
  1. Item a
  2. Item b
  3. Item c
  1. Item I
  2. Item II
  3. Item III
  1. Item i
  2. Item ii
  3. Item iii

Ordered list with reversed order

  1. Item 1
  2. Item 2
  3. Item 3

Unordered list: Disc, Square, Circle

  • Item 1
  • Item 2
  • Item 3
  • Item 1
  • Item 2
  • Item 3
  • Item 1
  • Item 2
  • Item 3

Nested lists

  • Item 1
    • Item 1.1
    • Item 1.2
  • Item 2
    • Item 2.1
    • Item 2.2
