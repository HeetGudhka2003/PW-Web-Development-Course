Block Level Elements and Inline Elements in HTML

The Block Level Element is the element which occupies the full width of the page. They are displayed in a new line. The examples of block level elements are p, h1, h2, h3, h4, h5, h6, div etc.

The Inline Element is the element which occupies only the required space. They are displayed in the same line. The examples of inline elements are span, em, strong, b, i, u, img etc.
